# Players

Players a princípio é uma API de estudo, 

Com ela você pode gerenciar seu grupo de amigos gamers, por exemplo.

* **Adicionar** um player,
* **Remover** um player,
* **Editar** um player,
* **Adicionar** games a esse player,
* **Remover** games de um player
